# Aluvery V1.0.0
Este é um aplicativo desenvolvido durante o curso de Jetpack Compose da Alura, focado na criação e gerenciamento de uma lista de produtos. Com ele, é possível visualizar uma variedade de produtos categorizados, cada um com sua imagem, nome e preço formatado de acordo com o padrão brasileiro.

## ✔️ Funcionalidades

- **Visualização de Produtos**: Exibe produtos categorizados como promoções, doces e bebidas com informações detalhadas.
- **Dados de Amostra**: Utiliza dados de amostra para exibição de produtos.
- **Componentes Customizados**: Implementa composables personalizados para itens de produto e layout da tela inicial.
- **Estilização de Tema**: Utiliza um tema personalizado para estilizar o aplicativo.
- **Formatação de Preço**: Apresenta preços dos produtos no formato de moeda brasileira.

## 🌟 Novidades na versão 1.0.0

- **HomeScreen**: Criação de uma tela inicial `HomeScreen` com seções de produtos.
- **Dados de Amostra**: Introdução de dados de amostra para produtos no arquivo `sampledata`.
- **Componentes de Produto**: Atualização do componente `ProductItem` para exibir dados dinâmicos.
- **Estilização**: Implementação de um esquema de cores personalizado e estilização dos componentes Composable.

## 🔨 Técnicas e tecnologias no projeto

- `Kotlin`: Linguagem de programação utilizada para desenvolver o aplicativo.
- `Jetpack Compose`: Biblioteca para construção de UIs em Kotlin.
- `Composables`: Uso de composables para criação de UIs reativas e declarativas.
- `State Management`: Gerenciamento de estados para atualização dinâmica da UI.
- `Preview Annotation`: Utilização da anotação `@Preview` para pré-visualizar composables durante o desenvolvimento.
- `Modelagem de Dados`: Definição de modelos de dados para produtos e utilização de listas para simular uma fonte de dados.
- `Extensões Kotlin`: Criação de funções de extensão para tarefas comuns, como a formatação de preços.
